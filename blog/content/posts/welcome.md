# Welcome to Linkdinger

This is your first post in your new blog. Edit this file or add new markdown files to `content/posts/` to get started.

## Features

- **Glassmorphism Design**: Beautiful frosted glass effects
- **Neubrutalism Accents**: Bold borders and shadows
- **Dark Mode**: Easy on the eyes
- **Fast**: Static generation for lightning-fast load times

## Writing Posts

Just create a new `.md` file in `content/posts/` with frontmatter:

```yaml
---
title: "Your Post Title"
date: "2026-02-19"
excerpt: "A short description"
tags: ["tag1", "tag2"]
publish: true
---
```

Then write your content in Markdown below.

## Images

Images are automatically uploaded to Cloudflare R2 when you paste them in Obsidian. The watcher daemon converts them to WebP and replaces local links with R2 URLs.

## Publish Control

Set `publish: false` to keep a post private (won't appear on the blog but will be backed up to GitHub).

Happy writing! 🚀
