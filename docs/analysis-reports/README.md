# Linkdinger — Analysis Reports

This directory contains analysis reports generated by the **Analysis Team (Gemini 2.5 Pro)**.

## Naming Convention

```
YYYY-MM-DD-[topic].md
```

## Examples

- `2026-03-05-trending-ui.md`
- `2026-03-05-test-coverage.md`
- `2026-03-05-auth-review.md`

## Rules

- Reports are **read-only output** from the Analysis Team
- Implementation Team reads these reports before coding
- Post-implementation reviews are saved here too
